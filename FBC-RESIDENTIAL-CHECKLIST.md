## üèóÔ∏è STRUCTURAL / BUILDING (FBC-R Chapter 3)

### Foundation (R403)
- [ ] Footing depth: Minimum 12" below undisturbed ground
- [ ] Footing width: Minimum 12" for 1-story, 15" for 2-story
- [ ] Reinforcement: #4 rebar minimum (2 bars continuous)
- [ ] Concrete strength: Minimum 2500 PSI footings, 3000 PSI slab
- [ ] Termite protection noted (FBC R318)
- [ ] Soil bearing capacity specified

### Slab (R506)
- [ ] Minimum 3.5" thick concrete slab
- [ ] Vapor retarder under slab
- [ ] Reinforcement per design (fibermesh or rebar)

### Wall Framing (R602)
- [ ] Stud size and spacing specified (typically 2x4 or 2x6 @ 16" O.C.)
- [ ] Header sizes shown for all openings
- [ ] Corner bracing / wall bracing noted
- [ ] Fire blocking at concealed spaces
- [ ] Top plate: Double 2x required

### Wind Design (FBC R301.2.1)
- [ ] Design wind speed specified (varies by county)
- [ ] Exposure category noted (B, C, or D)
- [ ] Wind-borne debris region compliance (if applicable)
- [ ] Hurricane straps/clips specified (Simpson H2.5A or equivalent)
- [ ] Impact-resistant openings OR shutters (if in HVHZ)

### Roof Framing (R802)
- [ ] Truss design by licensed engineer (if trusses used)
- [ ] Rafter size and spacing
- [ ] Ridge beam sizing
- [ ] Roof sheathing thickness (min 7/16" OSB or 1/2" plywood)
- [ ] Nailing schedule per wind zone
- [ ] Collar ties or ridge straps

## üö™ EGRESS & LIFE SAFETY (FBC-R Chapter 3)

### Emergency Escape Windows (R310)
- [ ] **Minimum opening area: 5.7 sq ft** (5.0 sq ft at grade)
- [ ] **Minimum opening height: 24"**
- [ ] **Minimum opening width: 20"**
- [ ] **Maximum sill height: 44" from floor**
- [ ] Required in ALL sleeping rooms
- [ ] Required in basements with habitable space

### Interior Doors
- [ ] Bedroom doors: Minimum 32" clear width
- [ ] Bathroom doors: Minimum 24" clear width
- [ ] All doors: Minimum 6'8" height

### Hallways (R311.6)
- [ ] Minimum width: 36"
- [ ] Minimum ceiling height: 7' (6'4" at beams)

## ü™ú STAIRS (FBC-R R311.7)

### Dimensions
- [ ] **Maximum riser height: 7¬æ"**
- [ ] **Minimum tread depth: 10"**
- [ ] **Minimum stair width: 36"**
- [ ] **Minimum headroom: 6'8"**
- [ ] Rise/run uniform throughout (max 3/8" variance)

### Handrails (R311.7.8)
- [ ] Required on at least one side
- [ ] Height: 34" to 38" from stair nosing
- [ ] Graspable profile (1¬º" to 2" diameter)
- [ ] Continuous full length of stairs
- [ ] Returns to wall or post at termination

### Landings (R311.7.6)
- [ ] Required at top and bottom
- [ ] Minimum depth: Equal to stair width (36")

## üõ°Ô∏è GUARDRAILS & DECKS (R312)

### Guardrails
- [ ] Required where walking surface >30" above grade
- [ ] **Minimum height: 36"** (residential)
- [ ] **Maximum baluster spacing: 4"** (sphere rule)
- [ ] No climbable elements

### Decks
- [ ] Ledger connection details (bolts to band joist)
- [ ] Joist hangers at ledger
- [ ] Flashing at ledger
- [ ] Post footings sized for load
- [ ] Beam/joist sizing per span tables

## üîå ELECTRICAL (NEC / FBC-R E3601+)

### General
- [ ] Service panel size adequate (100A min typical)
- [ ] Panel location accessible

### Outlets ‚Äî Spacing
- [ ] **Wall outlets: Maximum 12' apart**
- [ ] **Maximum 6' from any corner or opening**
- [ ] Kitchen counter: Outlet within 24" of each counter space
- [ ] Bathroom: At least one outlet within 36" of sink

### GFCI Required Locations (E3902)
- [ ] All kitchen countertop outlets
- [ ] All bathroom outlets
- [ ] Garage outlets
- [ ] Outdoor outlets
- [ ] Laundry area outlets
- [ ] Within 6' of any sink
- [ ] Unfinished basement outlets

### AFCI Required (E3902)
- [ ] All bedroom outlets and circuits
- [ ] Living areas (varies by jurisdiction)

### Smoke & CO Detectors
- [ ] Smoke: Inside each bedroom
- [ ] Smoke: Outside each sleeping area (hallway)
- [ ] Smoke: Each floor including basement
- [ ] CO detector: Each floor with fuel-burning appliance
- [ ] 120V hardwired with battery backup
- [ ] Interconnected (all alarm together)

### Lighting
- [ ] At least one switched light each room
- [ ] Exterior lights at entries
- [ ] Stairway lighting with switches at both ends

## üí® HVAC / MECHANICAL (FBC-R M1301+)

### Equipment
- [ ] Equipment sized per Manual J load calculation
- [ ] Equipment location accessible for service
- [ ] Clearances from combustibles

### Ductwork
- [ ] Duct sizing per Manual D
- [ ] Return air pathway from each room
- [ ] Ducts in unconditioned space insulated (R-6 min)

### Ventilation
- [ ] Bathroom exhaust fans (50 CFM min)
- [ ] Kitchen range hood or exhaust
- [ ] Dryer exhaust to exterior

### Combustion Air
- [ ] Adequate combustion air for gas appliances
- [ ] Make-up air for large exhaust fans

## üöø PLUMBING (FBC-P Chapter 4+)

### Fixtures
- [ ] Fixture count adequate per occupancy
- [ ] Accessible cleanouts
- [ ] Proper trap sizing

### Water Heater
- [ ] Seismic strapping (if required)
- [ ] T&P relief valve with discharge
- [ ] Drain pan (if in living space)
- [ ] Expansion tank (if closed system)

### Venting
- [ ] Each fixture properly vented
- [ ] Vent termination above roof
- [ ] Proper vent sizing

### Water Supply
- [ ] Adequate pipe sizing for fixture count
- [ ] Backflow prevention
- [ ] Shutoff valves at fixtures

## üî• FIRE PROTECTION (R302)

### Fire Separation
- [ ] Garage-to-house wall: 1/2" drywall minimum
- [ ] Garage-to-house ceiling: 5/8" Type X if room above
- [ ] Self-closing door from garage to house (20-min rated)
- [ ] No openings from garage to sleeping areas

### Exterior Wall Fire Separation
- [ ] <3' from lot line: 1-hour rated, no openings
- [ ] 3'-5' from lot line: 1-hour rated, limited openings
- [ ] >5' from lot line: No requirements

## üå°Ô∏è ENERGY CODE (FBC-Energy / IECC)

### Insulation (Climate Zone 2 ‚Äî most of FL)
- [ ] Ceiling: R-38 (or R-30 in cathedral)
- [ ] Wall cavity: R-13 or R-19 (2x6)
- [ ] Floor over unconditioned: R-13
- [ ] Slab edge: R-0 (not required in CZ2)

### Windows & Doors
- [ ] U-factor: 0.40 or less (FL Zone 2)
- [ ] SHGC: 0.25 or less

### Air Sealing
- [ ] Building envelope sealed
- [ ] Blower door test may be required

## üìê SITE / SETBACKS

### Verify Against Local Zoning
- [ ] Front setback compliant
- [ ] Side setbacks compliant
- [ ] Rear setback compliant
- [ ] Building height within limits
- [ ] Lot coverage within limits
- [ ] Impervious surface limits

### Flood Zone (if applicable)
- [ ] Base flood elevation determined
- [ ] Lowest floor above BFE
- [ ] Flood-resistant materials below BFE
- [ ] Utilities elevated or protected

## üìù COMMON REJECTION REASONS

1. **Missing egress windows** in bedrooms
2. **Stair dimensions non-compliant** (rise/run)
3. **Wind design not specified** or inconsistent
4. **Missing smoke/CO detector locations**
5. **GFCI not shown** at required locations
6. **Incomplete electrical plan** (outlets not counted)
7. **Header sizes missing** at window/door openings
8. **Garage separation not detailed**
9. **Energy code compliance not demonstrated**
10. **Site plan missing setback dimensions**
